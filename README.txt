----------------------------------------------------------
PROJECT OVERVIEW
----------------------------------------------------------

ClientFlow CRM Dashboard is a modern web dashboard UI/UX
design created using Figma for Customer Relationship
Management (CRM).

The dashboard enables businesses to manage leads,
clients, revenue, tasks, analytics, and recent deals
through a clean, intuitive, and responsive interface.
